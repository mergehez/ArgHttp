# Gradle

To always build from the latest commit with all updates. Add the JitPack repository:
```
repositories {
    maven { url "https://jitpack.io" }
}
```
And the dependency:

```
	dependencies {
	        compile 'com.github.mergehez:ArgHttp:master-SNAPSHOT'
	}

```
